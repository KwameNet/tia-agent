(*...*): Insert a comment section (S7-1200, S7-1500)
(*...*): Insert a comment section
Description
You can use the "Insert a comment section" instruction to add a comment section. The text
inside the parentheses "(*...*)" is handled as a comment.
Example
The following example shows how the instruction works:
SCL
(*This is a comment section.*)
