STP: Exit program (S7-1200, S7-1500)
STP: Exit program
Description
You use the "Exit program" instruction to set the CPU to STOP mode and therefore to ter-
minate the execution of the program. The effects of changing from RUN to STOP depend
on the CPU configuration.
